## Welcome to the Digital Dreamscape

HYPERPOP is a Y2K-inspired vaporwave blog built with [Eleventy](https://www.11ty.dev/), the simple static site generator. This project combines nostalgic aesthetics with modern web performance to create a unique digital experience.

### The Vision

We believe the web should be:

- **Fast** - Lightning-quick load times with static generation
- **Beautiful** - Stunning visuals that capture the Y2K spirit
- **Accessible** - Usable by everyone, respecting user preferences
- **Open** - Built with open-source tools and technologies

### The Tech Stack

This site is powered by:

- **11ty** - Static site generation
- **Nunjucks** - Templating engine
- **Vanilla CSS** - All visual effects with zero frameworks
- **Vanilla JavaScript** - Minimal, purposeful interactions
- **Netlify** - Hosting and deployment

### The Aesthetic

Our design philosophy draws inspiration from:

- Y2K era design trends
- Vaporwave and cyberpunk aesthetics
- CRT monitors and glitch art
- Neon colors and holographic effects
- Digital nostalgia
